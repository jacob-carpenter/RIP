Making New Friends Should be Easy

Core Beliefs

Forming new friendships requires a platform consisting of three core pillars: Search, Communication, and Event Planning

Assist the user in finding friends for any interest regardless of the niche, and in any location.

Provide a robust chat to allow users to become familiar with one another.

Allow users and groups to plan events to solidify friendships.

Friendships are Formed Based Upon a Few Shared Interests

Allow the user to enter any information they want to describe themselves or their groups.

By not prompting users for specific input but allowing users to specify any interests, the conscious decision to include an interest adds significant weight.

Making Friends is Different from Finding Romance

While a platform to find romance optimizes for one on one connections, a friend finding platform is severely limited by only catering to similar one on one relationships.

Friendships are predominantly formed through small interactions of individuals within a larger group.

To create a top platform for forming friendships, an organization must focus on both one on one and group connections.
